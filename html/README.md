This project is a web application built with the Laravel framework.
## Setup
Clone the repository:
```
git clone https://github.com/SophearithSaing/full-stack-test.git
```

Navigate to the laravel directory:
```
cd full-stack-test/html
```

Install the dependencies:
```
composer install
```

Start the development with server of your choice (I use XAMPP)
